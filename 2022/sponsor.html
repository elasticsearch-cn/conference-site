主办方

Elastic
Elastic 能使您在企业搜索、可观测性和安全等领域中，驾驭大规模的实时数据。Elastic 基于一个开源且开放的统一技术堆栈，构建了各种可在任何地方部署的解决方案，让您在任何类型的数据中，都能即刻得到可以付诸于行动的洞见——从查找文档，到监控基础设施，再到威胁狩猎，都能胜任。在全球范围内，有数以千计的公司/组织正在使用 Elastic 支持业务关键型系统，这些公司/组织包括思科、高盛、微软、Mayo 医学中心、美国国家航空航天局 (NASA)、纽约时报、维基百科和 Verizon等。Elastic 公司成立于 2012 年，已在 NYSE（纽约证券交易所）公开上市，股票代码为 ESTC。更多详情请参见 elastic.co/cn。
Elastic 中文社区
Elastic 中文社区专注于围绕 Elastic 开源项目：Elasticsearch、Logstash、Kibana、Beats 等及周边技术的交流与探讨。每年组织承办 Elastic 中国开发者大会和一二线城市的线上线下 Meetup 活动，邀请演讲嘉宾有来自 Elastic 官方、Google、百度、腾讯、阿里巴巴、字节跳动、360、ebay、微博、美团、58、苏宁等众多公司的技术专家，带来过众多精彩的分享，为中国广大的 Elasticsearch 开发者提供一个技术交流和学习切磋的地方，汇集业界众多的成功案例，集思广益，发散思维，促进社区和行业的进步。社区官网：https://elasticsearch.cn
极限科技
极限数据（北京）科技有限公司，简称极限科技，是一家专注于开源搜索与实时数据分析的软件公司。公司由前 Elastic 中国区首席技术顾问、Elastic 中文社区发起人兼主席曾勇 (Medcl）创建。旗下品牌极限实验室（INFINI Labs）致力于打造极致易用的数据探索与分析体验。自研产品及解决方案包括：Elasticsearch 多集群管理、企业级异地多活容灾、平台化数据治理等多种解决方案。更多详情请参见: https://极限科技.com

战略级赞助商

阿里云
阿里云（https://www.aliyun.com）创立于2009年，是全球领先的云计算及人工智能科技公司，为200多个国家和地区的企业、开发者和政府机构提供服务。阿里云在全球25个公共云地域开放了80个可用区，为全球数十亿用户提供可靠的计算支持。基于 Elasticsearch 及商业功能，阿里云检索分析服务 Elasticsearch 版兼容开源ELK功能，提供免运维全托管服务的弹性云搜索与分析引擎，致力于数据库加速、数据分析、信息检索、智能运维监控等场景服务；独有的云原生高性能内核、达摩院NLP分词、向量检索、智能运维、免费X-Pack高级商业特性等能力，全面提升企业应用效率，降低成本。 详请参见 https://www.aliyun.com/product/bigdata/elasticsearch
腾讯云
腾讯云 Elasticsearch Service（ES）是基于开源引擎打造的云端全托管 ELK 服务，集成 X-Pack 特性、独有高性能自研内核、QQ 分词、集群巡检、一键升级等优势能力，引入极致性价比的腾讯自研星星海服务器。助您轻松管理和运维集群，高效构建日志分析、运维监控、信息检索、数据分析等业务。

钻石赞助商

XSKY 星辰天合
星辰天合(北京)数据科技有限公司(简称“XSKY星辰天合”)是一家专注于软件定义基础架构业务的中国高新技术企业，基于软件定义的统一技术架构和混合云模式的数据管理方案，为政企客户提供以数据价值为核心的产品和服务体系，以满足日益扩大的海量数据的存储、管理、应用的需求，帮助政企客户实现海量数据的“汇-存-管-用”，提升数字化转型升级和创新的效能，夯实数字经济社会的基石。 公司成立于2015年，已成长为国内领先的数据基础设施技术平台提供商，产品获得了超1000家大型政企机构的大规模商业化部署。根据IDC等第三方权威报告，XSKY星辰天合是SDS中国市场TOP5中唯一一家专业厂商。
移动云
移动云隶属于中国移动通信集团公司，是中国移动面向政企、事业单位、开发者等客户推出的基于云计算技术、采用互联网模式、提供基础资源、平台能力、软件应用等服务的业务。移动云是建立在中国移动“大云”的基础上，自主技术研发而成的公有云平台，通过服务器虚拟化、对象存储、网络安全能力自动化、资源动态调度等技术，将计算、存储、网络、安全、大数据、开放云市场等作为服务提供，客户根据其应用的需要可以按需使用、按使用付费。

白金赞助商

Cloudera
Cloudera 是一家提供企业数据云的开源软件公司。Cloduera 的 CDH 大数据软件在金融，电信，制造业，互联网等行业有着广泛的使用用户群，其新一代的 Cloudera Data Platform（ CDP ）可以帮助企业实现从边缘计算到AI的大数据全生命周期的管理，在统一平台上进行多功能分析，消除孤岛并加速发现数据驱动的洞察力，共享数据体验，应用一致的安全治理和元数据管理，支持公有云，私有云和多云的混合云部署。2014年 Cloudera 进入中国以来，在中国发展了几百家客户，例如招商银行，浦发银行，光大银行，银联，上海大众汽车，永辉超市，中国联通等。今年10月，Cloudera 和阿里云在杭州云栖大会上联合发布 Cloudera 在阿里云上的 CDP 服务，让广大中小企业可以在阿里云的平台上实现 CDP 的大数据应用。
PingCAP
PingCAP 是业界领先的企业级开源分布式数据库企业，提供包括开源分布式数据库产品、解决方案与咨询、技术支持与培训认证服务，致力于为全球行业用户提供稳定高效、安全可靠、开放兼容的新型数据服务平台，解放企业生产力，加速企业数字化转型升级。
新智锦绣
新智锦绣科技（北京）有限公司（New Mind）于2018年开始与 Elastic 中国合作推广。公司团队主要人员基于多年在X86服务器、Linux操作系统、虚拟化技术、高性能计算分析、Kubernetes云平台、Hadoop大数据等领域从事企业级基础架构领域与开源数据技术服务的积累，目前已发展成为专注于提供开源软件技术架构演进的企业级实践服务：面向大数据搜索分析场景类的ELK服务，面向容器化数据技术创新类的K8S服务，面向开源软件技术教育类服务。所提供的技术服务领域主要围绕企业在面对开源软件技术驱动基础架构在快速演进迭代的变革背景而构建，核心价值是全方位地帮助大中型企业更高效地采用开源技术的持续性创新能力和更灵活地建设数据智能的多样性业务应用。围绕 Elasticsearch 开源技术生态提供服务，并与 Elastic 结为合作伙伴关系，双方共同响应客户在内容搜索、服务观测、安全防控等三类技术保障服务的需求。
StreamNative
StreamNative 是一家开源基础软件公司，由 Apache 软件基金会顶级项目 Apache Pulsar 创始团队组建而成，致力于围绕 Apache Pulsar 打造下一代云原生批流融合数据平台。
StreamNative 作为 Apache Pulsar 商业化公司，专注于开源生态和社区构建，致力于前沿技术领域的创新，创始团队成员曾就职于 Yahoo、Twitter、 EMC、腾讯、阿里巴巴等国内外知名公司。目前已获得 Prosperity7 Ventures 与华泰创新联合领投的 A 轮 2300 万美元融资，老股东红杉中国、源码资本持续加码。
智联猎头
智联猎头作为智联招聘旗下品牌，以满足企业中高端岗位的人才招聘需求为出发点，整合全球优质资源，覆盖数字经济、新一代信息技术、人工智能、金融、电子信息、装备制造、生物医药与健康、房地产/建筑等数十个行业，拥有规模庞大、覆盖面广的交付团队，致力于向企业输出专业化的招聘解决方案。
智联猎头目前拥有卓聘平台服务、招聘流程外包服务、高端人才猎寻服务三大类核心服务，具备高度定制化能力，能够为不同发展阶段的企业提供一站式招聘解决方案，同时为企业转型升级提供保障，全面增强企业核心竞争力！
智联猎头利用自身独特的资源优势、成熟的服务模式，以及多层次的产品，致力于打造高效、专业的一站式人才服务模式，让招聘更加美好！
